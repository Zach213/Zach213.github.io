TalkMode

Try on Chrome Web Store

About

TalkMode is a Chrome Extension that lets you control your browser with natural language voice commands. Born from my own experience with cubital tunnel syndrome and collaboration with the Special Technology Access Resource Center in Seattle, it makes web navigation accessible for those with limited mobility or repetitive strain injuries.

Details

Problem

Many users struggle with keyboard and mouse input due to injuries, disabilities, or conditions like cubital tunnel syndrome.

Solution

Natural language voice control for common web tasks — managing Gmail, scrolling social media, browsing websites.

Impact

Featured by Google Chrome Web Store. 10k+ Reddit views. Interest from university assistive technology departments.

Learnings

Voice recognition, accessibility design, rapid iteration from user feedback. 20+ production updates shipped.
