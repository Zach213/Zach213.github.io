About RUStewed

RUStewed is a meal-planning tool tailored for students, helping them manage their meals efficiently and within budget. The app provides recipe suggestions, cost estimations, and nutritional data.

My Contributions:

  • Designed the core features and UI for the app.
  • Collaborated with developers to implement real-time meal planning.
  • Conducted user testing to refine the app's features.

Key Features:

  • Recipe suggestions based on available ingredients.
  • Integration with local grocery stores for cost estimation.
  • Nutritional information for every recipe.
