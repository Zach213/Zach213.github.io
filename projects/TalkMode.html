About TalkMode

TalkMode is a Chrome Extension that empowers users to control their web browsing experience using natural language voice commands. Born from personal experience with cubital tunnel syndrome and collaboration with the Special Technology Access Resource Center in Seattle, TalkMode makes web navigation accessible for those with limited mobility or repetitive strain injuries.

Demo

Problem

Many users face challenges with traditional keyboard and mouse input due to injuries, disabilities, or conditions like cubital tunnel syndrome.

Solution

A Chrome Extension that enables natural language voice control for common web tasks like managing Gmail, scrolling social media, and browsing websites.

Impact

Featured by Google Chrome Web Store, gained 10k+ Reddit views, and attracted interest from university assistive technology departments.

Learning

Developed expertise in voice recognition technology, accessibility design, and rapid iteration based on user feedback with 20+ production updates.
