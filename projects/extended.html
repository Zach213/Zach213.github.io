About

What if you could just talk to a website you don't own and reshape how it works?

  • Hide YouTube spoilers
  • Make NYTimes look like Studio Ghibli
  • Turn lecture videos into flashcards

Extended is a browser overlay that builds real Chrome extensions through natural language. Don't wait for Perplexity or OpenAI to guess what use cases you care about — build it yourself, in minutes.

Why This Exists

Every modern agent is basically a Chrome Extension in disguise. Companies offering agentic browsers are just cherrypicking use cases where a specific workflow is done on a specific website — that'd be better solved by a one-off chrome extension.

I've built 7 extensions. This would've made me the fastest in the world at building them.

Details

Problem

Agentic browsers are slow, inaccurate, and non-editable. Building extensions traditionally requires hours in devtools.

Solution

Browser overlay that builds real Chrome extensions through conversation, with hot reloading.

Status

Sprint 2 of my bootstrapped incubator. Rolling out slowly with alpha testers for safety.

Tech

Local server with hot reloading, custom system prompts, accessibility permissions handling.
